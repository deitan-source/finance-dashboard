⚙ Connect to Google Sheets

Connect the dashboard to a shared Google Sheet so all team members see the same data in real time.

1
Open Google Apps Script
Go to script.google.com → click New project (top-left)
2
Paste the script code
Click Code.gs on the left → select all (Ctrl+A) → delete → paste the full content of the Code.gs file you received → click 💾 Save
3
Authorize the script (one time)
In the function dropdown next to ▶ Run, select test → click ▶ Run → click Review permissions → Allow
4
Deploy as Web App
Click Deploy → New deployment
Click the ⚙ gear icon → choose Web app
Execute as: Me  |  Who has access: Anyone
Click Deploy
5
Copy the Web App URL & paste below
Copy the URL ending with /exec and paste it in the field below.
